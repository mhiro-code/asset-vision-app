## v0.3.0 MVP: SBI iDeCoスクショ取込

- 日次データ登録に「SBI iDeCo取込」を追加
- 基準日はスクショ上部の「YYYY/MM/DD 現在」をOCRで取得し、取得できない場合のみ今日をセット。変更可能
- 資産グループは `iDeCo（固定）`
- Ctrl+V、ドラッグ&ドロップ、画像ファイル選択に対応
- 画像貼り付け後に評価額と基準日のOCRを自動実行
- OCR結果は編集可能。カンマあり／なし入力に対応
- 保存時は `date + asset_group` を一意キーとして登録
- 保存ルール：`asset_group=iDeCo`、`source_name=SBI iDeCo`、`source_type=screenshot_ocr`
- 画像は永続保存せず、プレビュー・OCR・確認用途のみで利用

OCRにはブラウザ上で動作する Tesseract.js CDN を利用しています。OCRが読み込めない場合も、画像プレビューを見ながら評価額を手入力して保存できます。


## v0.3.1 変更点

- SBI iDeCo取込のOCR対象に基準日を追加
- 「YYYY/MM/DD 現在」から取得した基準日を基準日に自動セット
- 基準日を取得できない場合のみ今日の日付を基準日にセット
- 保存時の `date` は登録作業日ではなく評価額の基準日として扱う。日付項目の名称は「基準日」に統一
- 評価額OCR失敗時と基準日OCR失敗時のメッセージを個別に表示

## v0.3.2 基準日表記への統一

- SBI iDeCo取込画面の日付項目名を「基準日」に統一
- OCR結果と保存内容を確認するエリアとして「読取結果・保存内容」を整理
- 保存時の重複判定・入力チェック文言も基準日基準に変更
- 評価額OCR失敗メッセージを「画像プレビューを見ながら手入力してください。」に変更


## v0.3.5 SBI iDeCo取込UI最終整理

- 画面を「画像」「読取結果・保存内容」「保存」の3ブロックに整理
- 「OCR完了」の常時表示を削除し、OCR失敗時のみメッセージを表示
- 「OCR再実行」を画像操作として「画像を変更」と同じグループに移動
- 保存ボタンを最後の操作として独立配置
- 資産グループは編集できない固定表示に整理

## v0.4.0 共通取込基盤 / セゾンNISA取込MVP

- SBI iDeCo取込で完成した処理を、`importProfiles` と `ocrProfile` を使う共通取込基盤へ整理
- 共通化対象：スクショ貼付、画像プレビュー、OCR、基準日OCR、評価額OCR、手修正、保存前チェック、重複確認、新規登録、上書き保存、一覧・グラフ・ダッシュボード更新、保存後初期化、トースト通知
- OCRは「画像をテキスト化する共通処理」と「プロファイル別抽出処理」に分離
- `sbi_ideco` と `saison_nisa` のOCR抽出プロファイルを追加
- セゾンNISA取込タブを追加
- セゾンNISA保存ルール：`asset_group=セゾンNISA`、`source_name=セゾン投信`、`source_type=screenshot_ocr`
- 日次データは実測値のみ保存し、欠落日・観測開始日前を0円や前日値で補完しない
- セゾンNISAは初回登録日の基準日を観測開始日として扱う

## 回帰テスト：SBI iDeCo / セゾンNISA 共通取込

1. 取込タブで SBI iDeCo または セゾンNISA を選択する
2. 貼付案内文と資産グループ固定表示が選択タブに応じて切り替わることを確認する
3. スクリーンショットを Ctrl+V、ドラッグ&ドロップ、または画像選択で取り込む
4. 同じ領域に画像プレビューが表示されることを確認する
5. OCRが自動実行され、基準日と評価額が読取結果・保存内容へ反映されることを確認する
6. 基準日・評価額を手修正できることを確認する
7. 基準日あり、評価額あり、評価額が数値解釈可能な場合のみ保存ボタンが有効になることを確認する
8. 新規保存し、日次データ一覧・資産推移グラフ・ダッシュボードが更新されることを確認する
9. 保存完了トーストが表示され、取込画面が初期状態へ戻ることを確認する
10. 同一基準日・同一資産グループで保存し、重複確認ダイアログが表示されることを確認する
11. キャンセル時は入力内容が保持されることを確認する
12. 上書き時は既存データが更新され、一覧・グラフ・ダッシュボード更新、トースト通知、画面初期化が実行されることを確認する

## 回帰テスト：取込タブ切替時の状態初期化

- SBI iDeCo取込で画像貼付・OCR実行後、セゾンNISA取込へ切り替える。
  - 画像プレビューが消え、貼付案内が表示されること。
  - 基準日が今日の日付に戻ること。
  - 評価額が空欄になること。
  - 保存ボタンが無効になること。
  - 資産グループ固定表示が「セゾンNISA（固定）」になること。
  - OCR状態・OCRメッセージ・保存完了メッセージ・重複確認ダイアログが残らないこと。
- セゾンNISA取込で画像貼付・OCR実行後、SBI iDeCo取込へ切り替えた場合も同じ挙動になること。
- 保存成功後の初期化と、取込タブ切替時の初期化が同じ基準で動作すること。
